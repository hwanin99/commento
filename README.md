<strong> Computer Vision 직무에서 중요한 역량은 크게 세 가지로 나눌 수 있습니다. </strong>
1. 기술적 전문성
   - 이미지를 처리하는 알고리즘 이해
   - AI 모델링과 OpenCV 등 필수 도구의 활용 능력
   - 코드를 체계적으로 관리하고 품질을 유지하는 능력
  
2. 문제 해결과 분석 능력
   - 기술을 단순히 구현하는 것을 넘어, 주어진 데이터를 분석하고 문제를 해결하는 사고 능력이 중요합니다.
   - 예를 들어, 2D 데이터를 3D로 변환하는 과제는 현실의 복잡한 문제를 단순화하고, 이를 해결할 수 있는 창의적 접근법을 요구합니다.
  
3. 협업과 프로젝트 관리 능력
   - Git을 사용한 협업, 테스트 주도 개발, 결과물의 시각화와 공유 등은 팀 단위로 진행되는 실제 업무에서 필수적인 요소입니다.

---
## 1주차  
<strong> <Git 활용 및 픽셀 단위 이미지 처리 실습> </strong>  
> Git을 활용하여 코드 관리를 익히고, 제공된 이미지 데이터를 픽셀 단위로 분석 및 처리하는 작업입니다. 이를 통해 협업 능력과 Computer Vision의 기본기를 다질 수 있습니다.  
> 1. Git으로 레포지토리를 생성하고, 브랜치 및 버전 관리 방법을 실습합니다.
> 2. 픽셀 데이터를 분석하여 특정 영역을 추출하거나 색상 정보를 조작하는 코드를 작성합니다.
> 3. 작업 결과를 정리하여 제출합니다.  
  
## 2주차  
<strong> <Unit Test 구성 및 2D -> 3D 변환 알고리즘 설계> </strong>   
> Unit Test를 통해 작성한 코드를 검증하고, 2D 이미지를 3D로 변환하는 알고리즘을 구현합니다. 이를 통해 코드 품질과 Computer Vision 기술의 실제 활용 능력을 배웁니다.  
> 1. Unit Test(GoogleTest/gMock)를 작성하여 픽셀 처리 코드의 정확성을 검증합니다.
> 2. OpenCV를 활용하여 2D 데이터를 3D로 변환하는 알고리즘을 설계합니다.
> 3. 변환 결과를 시각적으로 확인할 수 있도록 이미지 시각화 코드를 작성합니다.
  
## 3주차  
<strong> <AI 모델링 및 OpenCV 활용 결과물 시각화> </strong>  
> AI 모델을 학습시키고 OpenCV를 활용하여 이미지에서 객체를 탐지하거나 특정 패턴을 분석하여, 결과물을 시각적으로 도출합니다.  
> 1. 제공된 데이터를 사용해 AI 모델을 학습시키고, 이미지 분석 알고리즘을 설계합니다.
> 2. OpenCV로 모델 결과를 시각화하고, 최종적으로 처리 결과를 이미지로 출력합니다.
> 3. 모델 학습 과정과 결과물을 문서화하여 제출합니다.
  
## 4주차  
<strong> <희망하는 제품 또는 SW 선정 후 개발 프로젝트 기획 및 실행> </strong>   
> 지금까지 학습한 기술(Git, Unit Test, 2D->3D 변환, AI 모델링 등)을 종합적으로 활용하여 원하는 제품 또는 소프트웨어를 개발합니다.  
> 1. 프로젝트 주제를 선정하고, 개발 목표 및 주요 기능을 정의합니다.
> 2. 필요한 기술을 선택하고, 프로젝트 계획서를 작성하여 개발을 진행합니다.
> 3. 최종 결과물을 발표 자료로 정리하여 제출합니다.
